### 🔧 Firestore データ形式（purchases コレクション）

**新形式（`itemsV2: true`）**:
```
items: [{ name, qty, unitPrice, tax: '8%'|'10%', usage: string[], purpose }]
subtotal         // 税抜合計
totalPrice       // 税込合計
// 本申請後:
items2: [{..., actualUnitPrice, actualLineSub, actualLineTotal }]
subtotalActual
totalActual
```

**旧形式**（そのまま残り、読み取りは従来通り動作）:
```
items: [{ name, price, tax }]
totalPrice       // 単純合計
purpose          // 全体目的
// 本申請後:
items2: [{..., actualPrice }]
totalActual
```
